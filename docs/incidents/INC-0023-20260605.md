## Root Cause Analysis

Infrastructure team restarted Redis for routine maintenance without coordinating with the application team. No cache warming procedure existed. All traffic fell through to the database simultaneously.

## Resolution

Executed manual cache warming script. Developed automated cache warming procedure to run on Redis startup. Added Redis maintenance runbook requiring app team sign-off.
